# Welcome to WebNcov!

The website provides information on the situation of the COVID disease

## Prerequisite
#### For the server-side
- .NET core >= 3.1
- SQL Server >= 2017
#### For the cline-side
- NPM >= 6.13.4
- NODE >= v12.16.1
- REACT >= 16.13.1


## Install
**node**
Follow this [link](https://nodejs.org/dist/v12.18.0/node-v12.18.0-x64.msi)
> npm install
